# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What this is

A Quarto-based teaching website for **ETC4500/ETC5450 Advanced R Programming** at Monash University. The site is published to https://arp.numbat.space and hosted via GitHub Pages.

## Build commands

```bash
make preview   # Live-preview site locally (quarto preview)
make build     # Render site to docs/ excluding slide files (quarto render --profile noslides)
make clean     # Remove _site/, _freeze/, and slide caches
```

To render everything including slides: `quarto render`

To render a single file: `quarto render week5/index.qmd`

## Repository structure

- `week1/` through `week12/` — weekly content folders, each containing:
  - `index.qmd` — week overview page with learning outcomes
  - `slides.qmd` — slide deck (excluded from `make build` via `noslides` profile)
  - `*.R` — code examples and exercise scripts
- `assignments/` — assignment QMD files, rubrics, and solutions
- `docs/` — rendered output (do not edit directly; generated by Quarto)
- `diagrams/` — teaching diagrams (.graffle, .png, .pdf)
- `_freeze/` — Quarto cached computation results
- `_extensions/` — custom Quarto extensions

## Key configuration files

- `_quarto.yml` — main site config: navbar, theme, render list, output directory (`docs/`)
- `_quarto-noslides.yml` — profile that excludes `week*/slides.qmd` from rendering
- `course_info.R` — semester dates, weekly schedule, assignment metadata, helper functions (`show_assignments()`, `submit()`, `show_slides()`)
- `assignments.csv` — assignment due dates, marks, Moodle IDs, and file paths
- `numbat.scss` — custom SCSS with Monash branding (primary color: `#006DAE`)
- `renv.lock` — reproducible R package environment (R 4.5.3)

## Content conventions

- New weekly content goes in `weekN/index.qmd` and `weekN/slides.qmd`
- The navbar schedule menu in `_quarto.yml` must be updated when adding newly visible weeks
- Slides for weeks 1–4 use `format: presentation-beamer`; the Monash Beamer theme files (`beamerthememonash.sty`, `beamercolorthememonashwhite.sty`) are in each week folder
- Slides for week 5 onwards use `format: presentation-typst` (week 5 is the first converted; remaining weeks will follow)
- `execute: cache: false` is set globally — computation is not cached by default
- Two-space indentation, UTF-8 encoding (per `arp.Rproj`)
